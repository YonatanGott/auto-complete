# AutoComplete - Employees

## To install and run locally
MongoDB client should be installed locally.

### Server :
```js
cd server
```
and then
```js
yarn install or npm install
```
after installation
```js
npm run dev
```
### Client :
```js
cd client
```
and then
```js
yarn install or npm install
```
after installation
```js
npm run build
npm run dev
```


## To run with Docker:
```js
 docker-compose up
```
